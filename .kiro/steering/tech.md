# Technology Stack

## Build & Runtime
- **Python**: Primary runtime (via `uvx` package manager)
- **uv**: Python package manager for dependency management

## Key Tools & Services
- **AWS Diagram MCP Server**: Integration for AWS architecture visualization
- **Git**: Version control system
- **MCP (Model Context Protocol)**: Framework for extending capabilities

## Development Environment
- **Windows**: Primary development platform
- **PowerShell/CMD**: Shell environment

## Common Commands
- `uvx --version`: Check uvx installation
- `uvx awslabs.aws-diagram-mcp-server`: Run AWS diagram server
- Git operations for version control

## Configuration
- `.claude/settings.local.json`: Local Claude settings and permissions
- `.kiro/steering/`: Steering rules and guidelines
- `.kiro/specs/`: Feature specifications
