# Product Overview

**Kiro** is an AI-powered IDE assistant designed to help developers write, test, and debug code more effectively.

## Purpose
Kiro provides intelligent code assistance, project analysis, and automation capabilities to enhance developer productivity and code quality.

## Key Characteristics
- Autonomous and supervised operation modes
- Context-aware code analysis and recommendations
- Integration with development workflows
- Support for multiple programming languages and frameworks
- Git-aware operations and version control integration
